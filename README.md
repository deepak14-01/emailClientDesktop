# Desktop Email Client Project

Built using JavaFX, JavaMail, SceneBuilder, IntelliJ.

## How to run
Requirements: JDK 11


 - Download the **.jar** file along with the **bash file**(.sh/.bat) [here](https://github.com/deepak14-01/Desktop-Email-Client-Java/new/main?readme=1). Open the bash/batch script to run the jar file with the prefixed configuration.
(Oracle has made it harder to run JavaFX jar files).
